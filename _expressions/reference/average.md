## AVERAGE

Returns the numerical average value in a dataset, ignoring text.

### Parameters

`var_args_values` Number (__required__) - values to use when calculating the average value.

### Returns

Number

### Examples

~~~
AVERAGE(1, 2, 3)

// returns 2
~~~
