## PROPER

Capitalizes each word in a string (title case).

### Parameters

`value` String (__required__) - String to capitalize words within.

### Returns

String

### Examples

~~~
PROPER("fulcrum mobile data collection")

// returns Fulcrum Mobile Data Collection
~~~
