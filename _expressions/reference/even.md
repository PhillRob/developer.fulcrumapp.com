## EVEN

Returns number rounded up to the nearest even integer.

### Parameters

`value` Number (__required__) - Number to round.

### Returns

Number

### Examples

~~~
EVEN(5.2)

// returns 6
~~~
{: .language-js}


~~~
EVEN(528.2172814)

// returns 530
~~~
{: .language-js}
