### Parameters

`value` Number (__required__) - Number to round.

### Returns

Number

### Examples

{% highlight js %}
EVEN(5.2)

// returns 6
{% endhighlight %}


{% highlight js %}
EVEN(528.2172814)

// returns 530
{% endhighlight %}
