### Parameters

`value` Number (__required__) - The number for which to calculate the square root.

### Returns

Number

### Examples

{% highlight js %}
SQRT(5)

// returns 2.23606797749979
{% endhighlight %}


{% highlight js %}
SQRT(-9)

// returns NaN
{% endhighlight %}
