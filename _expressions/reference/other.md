## OTHER

Returns the current 'Other' value for a choice field or classification field

### Parameters

`field` Object (__required__) - The choice field or classification field variable

### Returns

String - the current 'Other' value

### Examples

~~~
OTHER($choice_field)

// returns undefined
~~~
