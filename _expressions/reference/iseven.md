### Parameters

`value` Number (__required__) - The value to be verified as even.

### Returns

Boolean

### Examples

~~~
ISEVEN(4)

// returns true
~~~
{: .language-js}


~~~
ISEVEN(5)

// returns false
~~~
{: .language-js}


~~~
ISEVEN(4.12)

// returns true
~~~
{: .language-js}
