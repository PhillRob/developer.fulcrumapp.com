## LCM

Returns the least common multiple of one or more integers.

### Parameters

`var_args_values` Number (__required__) - Value(s) to consider to calculate the least common multiple.

### Returns

Number

### Examples

~~~
LCM(2, 5)

// returns 10
~~~
{: .language-js}


~~~
LCM(5, 22, 12)

// returns 660
~~~
{: .language-js}
