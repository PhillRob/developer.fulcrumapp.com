## ACOSH

Returns the inverse hyperbolic cosine of a number.

### Parameters

`value` Number (__required__) - The value for which to calculate the inverse hyperbolic cosine. Must be greater than or equal to 1.

### Returns

Number

### Examples

~~~
ACOSH(7)

// returns 2.6339157938496336
~~~
