## ACOS

Returns the inverse cosine of a value, in radians.

### Parameters

`value` Number (__required__) - The value for which to calculate the inverse cosine. Must be between -1 and 1, inclusive.

### Returns

Number

### Examples

~~~
ACOS(0.8)

// returns 0.6435011087932843
~~~
