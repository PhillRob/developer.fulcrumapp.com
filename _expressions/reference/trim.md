## TRIM

Strips the leading and trailing spaces from a string.

### Parameters

`value` String (__required__) - argument

### Returns

String

### Examples

~~~
TRIM("  Fulcrum is cross-platform ")

// returns Fulcrum is cross-platform
~~~
