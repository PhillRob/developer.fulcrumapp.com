### Parameters

`show` Boolean (optional)  [default = true] - Show full error messages. This can help diagnose problems in expressions on the mobile device.

### Returns

Boolean

### Examples

{% highlight js %}
SHOWERRORS()

// returns true
{% endhighlight %}
