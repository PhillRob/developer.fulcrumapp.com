### Parameters

`var_args_values` Number (__required__) - Value(s) to consider to calculate the greatest common divisor.

### Returns

Number

### Examples

```js
GCD(5, 9)

// returns 1
```


```js
GCD(14, 21, 42)

// returns 7
```
